# Automação e Análise de Dados de Vendas

Este projeto foi desenvolvido para solucionar inconsistências comuns em bases de dados empresariais. O script implementa uma rotina de limpeza automática de valores ausentes e gera indicadores de performance estratégicos para o suporte à tomada de decisão.

## Tecnologias e Bibliotecas

*   **Python**: Linguagem principal utilizada para o desenvolvimento da lógica de automação.
*   **Pandas**: Biblioteca aplicada na estruturação de DataFrames, manipulação de dados e operações de agrupamento.
*   **NumPy**: Utilizado para o processamento matemático e suporte técnico a valores nulos (NaN).
*   **Matplotlib**: Implementação de visualizações gráficas para análise detalhada de faturamento.

## Funcionalidades e Metodologia

*   **Tratamento de Dados (Data Cleaning)**: Identificação e correção de dados faltantes através de métodos estatísticos de média e mediana, garantindo a integridade da base.
*   **Cálculo de Indicadores**: Processamento automático de métricas financeiras, incluindo Faturamento Total e Ticket Médio.
*   **Visualização de Resultados**: Geração de gráficos para análise comparativa de performance entre categorias de produtos.
*   **Exportação de Relatórios**: Consolidação dos dados processados em arquivos de formato CSV para uso em ferramentas externas.
